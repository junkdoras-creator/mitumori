# 02. 製本手順書

製本方法を事務所として統一し、パート側のミス（データ取り違え・上書き・並び間違い）を防ぐ。

## 共通ルール

- **別表一は青紙**で印刷する。
- **概況書は片面印刷**。
- **印刷元を統一する**：各ソフトから個別出力するとデータ取り違え・上書き事故が起きやすいため、
  原則「**所長報告チャットのデータ**」＋「**メール詳細**」を印刷元とする
  （ベースは達人の「電子申告完了報告書」＋「固定資産台帳」）。

## 全体の印刷順（参考）

法人税申告書 → 固定資産台帳 → 決算書 → 内訳書 → 法人税メール詳細 → 消費税申告書 → 消費税メール詳細 → 概況書 → 税務代理権限証書 → 地方税申告書 → 地方税メール詳細

> この並びをベースに、下記の「束」ごとにホチキス留めする。

## 束分け・ホチキス留め

封入は **束①→②→③→④→⑤→納税額一覧** の順。

| 束 | 内容 | 留め方 | 備考 |
|---|---|---|---|
| **束①** 法人税一式 | 法人税申告書（別表一＝青紙）→ 固定資産台帳 → 決算書 → 内訳書 → 税務代理権限証書 → 法人税メール詳細 | **二個留** | 別表一は青紙 |
| **束②** 概況書 | 概況書 | **ホチキスなし** | **片面印刷** |
| **束③** 消費税 | 消費税申告書 → 消費税メール詳細 | **左上 一個留** | |
| **束④** 地方税：都道府県 | 都道府県税申告書 → 地方税メール詳細（都道府県分） | **左上 一個留** | |
| **束⑤** 地方税：市町村 | 市町村税申告書 → 地方税メール詳細（市町村分） | **左上 一個留** | |
| 納税額一覧 | 納付税額一覧表 | 留めなし | 最後に添付 |

## 地方税が複数自治体・23区外の場合

- **自治体ごとに束を作る**（束④都道府県／束⑤市町村をそれぞれ自治体数だけ用意）。
- 申告先自治体はフォームの「申告先自治体（複数・23区外は必ず明記）」で確認する。

## 確認ポイント（事務所で最終確定する）

- ⚠️ **税務代理権限証書の位置**：全体の印刷順では「概況書の後」と記載があるが、本手順書では束①法人税一式（法人税メール詳細の前）に含める前提で配置している。最終確定が必要。
- ⚠️ **地方税メール詳細の扱い**：都道府県・市町村で1枚ずつ分けるのか、まとめて1枚かを確定する（本手順書は束ごとに付ける前提）。
