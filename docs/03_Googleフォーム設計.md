# 03. Googleフォーム設計（依頼の入口）

必須項目＋選択式で、記入漏れ・表記ゆれを構造的に防ぐ。**★＝必須**。

## 基本情報

| 項目 | 形式 | 選択肢・補足 |
|---|---|---|
| 顧問先名（達人コード）★ | 記述（短文） | コードも併記してもらう |
| 依頼者（社員名）★ | プルダウン | 社員リストから選択。※同一Google Workspaceなら「メールアドレスを自動収集」設定で代替可（メンテ不要・確実） |
| 対象事業年度・期 ★ | プルダウン×2 | 年度（例：2024〜）＋ 月（1〜12月の決算期） |
| 依頼種別 ★ | チェックボックス | □電子申告 □製本 □郵送 |

## ① 電子申告（要の場合）

| 項目 | 形式 | 選択肢 |
|---|---|---|
| 期限 ★ | 日付 | |
| 対象税目 ★ | チェックボックス（複数選択） | 法人税 ／ 消費税 ／ 都道府県 ／ 市町村 |
| 申告先自治体 | 記述 | **都道府県・市町村を選んだ場合は必須**。複数・23区外は必ず明記 |
| 添付書類の有無 | ラジオ＋記述 | なし ／ あり（具体的に：＿＿） |
| 納付情報の要否 | ラジオ | 不要 ／ ペイジー番号発行 ／ 納付情報発行依頼まで |

## ② 製本（要の場合）

| 項目 | 形式 | 選択肢 |
|---|---|---|
| 期限 | ラジオ＋日付 | 日付指定（＿＿）／ いつでも |
| 特記事項 | ラジオ＋記述 | なし ／ 元帳添付 ／ 議事録添付 ／ その他（＿＿） |

## ③ 郵送（要の場合）

| 項目 | 形式 | 選択肢 |
|---|---|---|
| 期限 | ラジオ＋日付 | 日付指定（＿＿）／ いつでも ※ラベルは①②と統一 |
| お届け先 | ラジオ＋記述 | 法人住所 ／ 社長自宅 ／ 別住所（＿＿） |
| 郵送方法 | ラジオ | エコ配 ／ 青レタパ ／ 赤レタパ |
| 同封物 | チェックボックス＋記述 | 納付書 ／ 返送資料 ／ その他（＿＿） |

## Googleフォーム構築時の注意

- **要/不要のセクション分岐**：「依頼種別」をチェックボックスではなく **ラジオ（要/不要）にした項目ごとのセクション**にすると、Googleフォームのセクション移動（条件分岐）が使える。
  - 例：「製本は必要ですか？（要/不要）」→ 要なら製本セクションへ、不要なら次へスキップ。
- **対象税目→申告先自治体の分岐の制約**：対象税目は複数選択（チェックボックス）のため、Googleフォームでは選択内容に応じた自動分岐ができない。
  → 回避策として「申告先自治体」は**常に表示**し、説明文に「都道府県・市町村を選んだ場合は必須／複数・23区外は明記」と記載する。
- **回答の自動転記**：［回答］タブ →［スプレッドシートにリンク］で台帳（[04](04_台帳設計.md)）に自動蓄積する。
- 必須（★）の項目は、フォームの「必須」トグルを必ずON。
