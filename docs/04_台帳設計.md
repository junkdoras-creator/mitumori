## プルダウン設定（データの入力規則）

- 申告ステータス：`不要, 依頼中, 完了`
- 製本ステータス：`不要, 依頼中, 完了`
- 郵送ステータス：`不要, 依頼中, 確認待ち, 完了`

## 全体ステータスの数式案

3タスクがすべて「完了」または「不要」なら「完了」、それ以外は「進行中」。
（申告ステータス＝T列、製本ステータス＝W列、郵送ステータス＝AB列の例。実際の列に合わせて調整）

```
=IF(AND(OR($T2="完了",$T2="不要"),OR($W2="完了",$W2="不要"),OR($AB2="完了",$AB2="不要")),"完了","進行中")
```

## 条件付き書式（色分け）

- ステータスが **依頼中** … 黄系で塗る
- ステータスが **確認待ち** … 橙系で塗る（担当者の確認が必要なことを強調）
- ステータスが **完了** … 緑系で塗る
- **期限が今日以前 かつ 未完了** の行 … 赤系で強調（例：`=AND($G2<TODAY(), $申告ステータス<>"完了")`）
- ステータスが **不要** … グレーで塗る（対象外を一目で）

## 完了報告の運用

郵送タスク完了時に、パートが次の3点を必ず記録する（やり忘れ・控え漏れ防止）。

- 郵送日
- 追跡番号（レタパ・書留など追跡可能な方法のとき）
- 控え保管✓
